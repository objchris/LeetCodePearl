# 动态规划

动态规划的思路是将问题划分为一个个小的问题，先解决子问题，再依据子问题的结果逐步解决大问题。

动态规划四元素：
1. 状态：需要思考在动态规划过程中需要保存什么状态
2. 初始化：定义完状态后，对某些特殊状态进行初始化
3. 转移方程：在于如何将问题分解，没有放之四海而皆准的计算动态规划的解决方案
4. 结果

动态规划一般借助表格来辅助思考，每个单元格中的值通常就是要优化的值。所以在使用动态规划时，多画画表格。
